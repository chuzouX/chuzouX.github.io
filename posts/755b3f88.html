集合的概念（一）

我们已经接触过一些集合：

2.圆的定义：平面内到定点的距离等于定长的点的集合.

集合的定义

集合定义的理解

  1. 是一定范围内的确定的对象；
  2. 是不同的对象；
  3. 是这些对象的全体.

归纳总结
一般地， 我们把研究对象统称为元素.
通常用小写拉丁字母a,b,c,…来表示.
我们把一些元素组成的总体叫做集合(简称为集).
通常用大写拉丁字母A,B,C,…来表示.
问题：组成集合的元素一定是数吗？
组成集合的元素可以是物、数、图、点等，它具备怎样的性质呢？

集合中元素的性质

  1. 所有的“帅哥”能否构成一个集合？由此说明什么？

    不能. 其中的元素不确定

集合中的元素是确定的

​ “帅”是一个含糊不清的概念，具有相对性，多么“帅”才算“帅”？没有明确的标准，也就是说，是一些不能够确定的对象．因此，不能构成集合．

  1. 由1,3,0,5,︱-3 ︳这些数组成的一个集合中有5个元素，这种说法正确吗？

    不正确.集合中只有4个不同元素1，3，0，5 .

    集合中的元素是互异的

  2. 高一（5）班的全体同学组成一个集合，调整座位后这个集合有没有变化？

    集合没有变化

    集合中的元素是没有顺序的

    两个集合中，元素完全一样，则称两集合相等.

通过以上的学习你能给出集合中元素的特性吗？
确定性、互异性、无序性

元素和集合的关系

已知下面的两个实例：
（1）用A表示高一(3)班全体学生组成的集合.
（2）用a表示高一(3)班的一位同学，b表示高一(4)班的一位同学.

思考：那么a，b与集合A分别有什么关系?
a是集合A中的元素,
b不是集合A中的元素

归纳总结

元素a与集合A的关系
如果a是集合A的元素，就说a属于集合A，记作a∈A ；
如果a不是集合A中的元素，就说a不属于集合A，记作a∉A.
属于符号和不属于符号具有方向性，左边是元素右边是集合。

【总结提升】由于元素完全相同的两个集合相等，而与列举的顺序无关，因此集合可以有不同的列举方法.例如，例1（1）可以表示为A={9,8,7,6,5,4,3,2,1,0}

描述法
思考深化

由于小于10的实数有无穷多个，而且无法一一列举出来，
因此这个集合不能用列举法表示．
但是可以看出，这个集合中的元素满足性质：
（1） 集合中的元素都小于10.
（2） 集合中的元素都是实数．
这个集合可以通过描述其元素性质的方法来表示，
写作: {x∈R|x<10}
我们可以把奇数集合表示为{x|x=2k+1,x∈Z}
还可以把奇数集合表示为{x|x=2k-1,x∈Z}
又如所有偶数的集合怎样表示？
{x∈Z|x=2k,k∈Z}

描述法：用这个集合所含元素的共同特征表示集合的方法．

注意：如果从上下文的关系来看， x∈R ，x∈Z是明确的，那么x∈R ，x∈Z 可以省略，只写元素x.
例如 { x∈R | x<10 }= { x | x<10 }
{ x∈Z | x=2k, k∈Z} = { x | x=2k, k∈Z}

归纳升华

思考：你能说出列举法和描述法的优缺点吗？

优点 缺点
列举法 直观、明了 不易看出元素所具有的属性，且有些集合不能用列举法表示
描述法 把集合中元素所具有的性质描述出来，具有抽象性、概括性、普遍性的特点 不易看出集合的具体元素

化学

电解质

1.电解质

在水溶液或熔融态下能够导电的化合物

2.电解质导电的条件

（1）一般来说，对于NaOH、KCl等可溶性的碱和盐，在水溶液和熔融状态都可以导电；
（2）对于难溶性的碱、盐、金属氧化物，如Mg(OH)2、BaSO4、Fe2O3，因为这些物质难溶于水，所以它们只能在熔融状态才可以导电；
（3）像HCl、H2SO4等酸只能在水溶液才可以导电。

注意
电解质与导电性之间没有必然关系：
（1）电解质不一定都能导电：如NaCl固体、HCl气体；
（2）能导电的物质不一定是电解质：如石墨、金属单质、溶液。

3.非电解质

在水溶液和熔融态下均不导电的化合物

注意
（1）电解质在“水溶液”里或“熔融状态”下能导电，二者具备其一即可；
（2）非电解质在“水溶液”里和“熔融状态”下不能导电，二者需同时满足；
（3）电解质和非电解质都是化合物，单质和混合物既不是电解质，也不是非电解质；
（4）电解质与非电解质的本质区别是看化合物自身在熔融状态下或在水溶液中能否发生电离，而溶液的导电只是一个表面现象。

（5）AgCl、BaSO4等难溶盐在水中的溶解度极小，不能导电，但这些物质在熔融状态下能导电，在水中溶解的部分也能完全电离，故属于电解质。
（6 )溶液导电性与自由移动的离子浓度有关，离子浓度越大，导电能力越强。

思考与交流

（1）金属Cu、稀盐酸都能导电，请问金属Cu、稀盐酸属于电解质吗？
都不属于，因为金属Cu是单质，稀盐酸是混合物，两者都不是化合物。
（2）判断盐酸、硫酸、稀硝酸、烧碱溶液是否为电解质或非电解质？
盐酸、稀硝酸、烧碱溶液为混合物，既不是电解质，也不是非电解质；硫酸为电解质。
（3）CO2、NH3的水溶液能导电，那么CO2、NH3属于电解质吗？
都不属于，因为电解质必须是自身能电离成自由移动离子的化合物。
（4）“一个物质如果不是电解质，就是非电解质”这种说法是否正确？
错误，如单质和混合物既不是电解质也不是非电解质。

若给出一个物质（我们不知道它溶于水或熔融状态下能否导电），怎么判断其是否为电解质？

电解质溶于水或受热融化时形成自由移动离子的过程。

电解质溶于水时是由于水分子的作用而电离熔融状态下是由于热能破坏化学键而电离成自由移动的离子都不是因为通电而电离

地球的形成与演化

地球的形成与演化简史

距今4570百万年地球形成,逐步演化至今。

原始地球的形成
距今4570百万年，地球形成。
46亿年以前，星云盘内物质经碰撞吸积，原始地球形成。

原始地表的形成
距今4570百万年，地表随之形成。
密度大的物质逐渐聚集到地球的中心，形成了地核，密度小的物质向上形成了地幔和地壳。

原始海洋的形成
距今4150年，海洋形成
广泛发生的火山喷发，释放出水蒸汽，二氧化碳等气体，构成了原始的大气层。 随地表温度降低水汽凝结降雨， 汇聚到地面形成了原始海洋。

原始生物的形成
距今3850百万年，地球生命出现。
原始生命物质形成于大约距今38.5亿年前的原始海洋。距今约5.4亿年前地球出现生命大爆发。180万 年前人类诞生。

知识小结

地球的演化过程

距今4570百万年地球形成，逐步演化至今。

生物演化(植物)
地球生物演化经历了从低级到高级，从简单到复杂的过程。
从空间分布来看，生物演化经历了由海洋向陆地扩 展的过程。

生物演化(动物)
地球生物演化经历了从低级到高级，从简单到复杂的过程。
从空间分布来看，生物演化经历了由海洋向陆地扩 展的过程。

生物演化
在生物演化过程中，伴随着一些生物的衰退和灭亡，是另一些生物的出现和兴盛。如恐龙与哺乳动物

海陆变迁
地质历史上曾经发生过多次海陆变迁，大约3亿~2亿年前形成泛大陆。大约距今2亿年前。泛大陆分裂形成两个大陆(冈瓦纳大陆和劳亚大陆)，约6500万年前，七大洲和四大洋的轮廓初步显现，目前已形成七大洲，四大洋的海陆格局。

构造运动
概念:由地球内部力量引起的地壳乃至岩石圈的变位、变形
在地球历史上曾发生过多次构造运动，构造运动导致了地形和海陆的变迁，对气候的变化，生物的演化。以及一些矿物质的形成具有重要影响。
发生在中生代的构造运动导致了泛大陆的解体;
发生在晚新生代的构造运动导致了青藏高原和喜马拉雅山的形成。

青藏高原的形成及影响

矿产形成
地质历史上出现了多个重要成矿期，如前寒武纪铁矿成矿期，古生代后期煤炭成矿期(重要成煤期)，中生代煤炭成矿期(主要成煤期)

矿产资源是经济和社会可持续发展的重要物质基础，是综合国力的重要组成部分。
矿产，泛指一切埋藏在地下(或分布于地表的、或岩石风化的、或岩石沉积的)可供人类利用的天然矿物或岩石资源。矿产可分为金属、非金属、可燃有机等类别，是不可再生资源。那么，矿产资源是如何形成的呢？

伴随着构造运动、海陆变迁、生物演化和气候变化，地质历史上出现了多
个重要的成矿期，如
前寒武纪铁矿成矿期（距今25亿至23亿年前后），
大量的铁、金、镍、铬等矿藏出现在这一时期的地层中；
古生代后期煤炭成矿期；
中生代煤炭成矿期等。

煤炭：暖湿的陆生环境 石油：温暖的浅海环境
